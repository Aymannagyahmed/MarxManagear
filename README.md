

## Introduction

MarxManager is a CRUD PHP web application that allow users to manage their Favorite bookmarks that includes adding ,removing, editting and deleting using:

- postgresql
- Laravel 7
- php
- bootstrap 4
- Axios

## Instlation

- composer install
- npm install
- php artisan migrate
- php run serve
